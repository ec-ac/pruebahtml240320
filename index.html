Abstraccion de hardware a Software

una vista a como se simplifica el proceso de crear un sistema de computo

Ernesto Alaniz

Parte Fundamental

La Abstraccion de hardware a software es una estrategia que se utiliza para simplficar multiples aspectos importantes de la creacion y desarollo de sistemas de computo, incluyendo:

  • La creacion de sistemas operativos
    • (mayormente aspectos como el kernel)
  • reforzar y simplificar el entendimiento de la forma en que la hardware funciona de forma fundamental
  • reforzar la comprension de como fundionan las computadoras a un nivel mas alto

Concepto

aqui se puede ver como una señal electrica fisica se puede convertir a informacion de "1" o "0" dependiendo de la intensidad, esto es un concepto fundamental para cualquier computadora

Proceso

El proceso de abstraccion de hardware a software es un proceso que se lleva a cabo en multiples pasos, los cuales son:

Señal electrica a informacion digital Informacion digital a instrucciones de maquina Instrucciones de maquina a lenguaje de bajo nivel Lenguaje de bajo nivel a lenguaje de alto nivel Lenguaje de alto nivel a software

Conclusión

La abstraccion de hardware a software es sumamente importante, ya que toca aspectos esenciales para los programadores, incluyendo:

  1. La forma en que se crean los sistemas operativos
  2. La forma en que se crean los lenguajes de programacion
  3. La forma en que se crean los programas de computadora
  4. un proceso fundamental para el desarollo de sistemas de computo
  5. un proceso que se lleva a cabo en multiples pasos, los cuales son fundamentales para el entendimiento de como funcionan las computadoras a un nivel mas alto
